# KENUXA CORE

The central intelligence, memory, orchestration, automation, authentication, and infrastructure platform for the KENUXA ecosystem.

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000/dashboard`.

## Setup

1. Copy `.env.example` to `.env.local`.
2. Create a Supabase project.
3. Run `supabase/schema.sql`.
4. Add Supabase, JWT, and Groq environment variables.
5. Deploy to Vercel.

## Philosophy

CORE is not REACH, VOICE, or Academy. Those products stay separate and connect through APIs, events, shared auth, shared memory, and webhooks.
